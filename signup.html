signup

signing up for a trash.town account is a little different than other websites. instead of a username and password, you generate a public key and a private key. the public key will stay on trash.town while the private key stays on your own computer, hidden away.

when you connect to trash.town, you will use a tool called SSH. SSH will use your private key to authenticate your login by checking to see if the public key on trash.town correlates with your private key on your own computer.

the instructions on this page will guide you through all of this, so don't worry if this is confusing!

requirements

to sign up for a trash.town account, you need the following items:

  • a unix-like environment, such as, but not limited to:
    • ubuntu
    • windows subsystem for linux
    • macOS
    • scoop.sh
  • SSH

preparing private and public ssh keys

ssh keys help prove who you are when you are logging into a server. this section will guide you through generating the ssh keys required.

to prepare private and public ssh keys

  1. run ssh-keygen -b 4096 -t rsa -f ~/.ssh/trash.town
  2. press enter, if you don't require a password for access to the key
  3. press enter again to confirm

what did that do?

this generated two things:

  • a ~/.ssh/trash.town key, which is your private key
  • a ~/.ssh/trash.town.pub key, which is your public key

what do i do with these keys?

you will need to send the public key to trash.town admins, so the trash.town ssh server can check if it is you when you ssh into trash.town

how do i send the public key?

try typing cat ~/.ssh/trash.town.pub

a long list of letters and numbers will show up on your terminal.

copy and paste this collection of text and send it to signup@trash.town.

be careful!

make sure you don't send your trash.town key (without the .pub at the end), because this is your super secret private key

if the steps above didn't work for you, have a look at tilde.town's guide or tilde.town's key generator
