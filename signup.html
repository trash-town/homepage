signup

signing up for a trash.town account requires you to generate a public key and a private key using a tool called SSH.

this page consists of the following sections:

  • conventions used on this page
  • requirements
  • generating your public and private ssh keys
  • sending your public ssh key to trash.town

conventions used on this page

  • note: a note signifies additional information.
  • tip: a tip signifies an alternative procedure.
  • warning: a warning signifies that data loss or a security breach could occur.
  • bolded items: bolded items that are not notes, tips, or warnings signify important items or details.

requirements

  • a unix-like environment, such as:
    • ubuntu
    • windows subsystem for linux
    • macOS
    • scoop.sh
  • SSH

generating your public and private ssh keys

public and private ssh keys are used when logging into an ssh server, without these keys, a user cannot log in.

this section will guide you through generating your public and private ssh keys.

note: the procedures in these steps should be run inside of a unix-like environment of your choice.

to generate your public and private ssh keys

  1. run ssh-keygen -b 4096 -t rsa -f ~/.ssh/trash.town
  2. press enter, if you don't require a password for access to the key
  3. press enter again to confirm

note: this generated two things:

  • a ~/.ssh/trash.town key (trash.town), which is your private key
  • a ~/.ssh/trash.town.pub key (trash.town.pub), which is your public key

sending your public ssh key to trash.town

this section will help you access your generated public key, so you can copy and paste it into an email.

warning: before you continue, make sure you don't send your ~/.ssh/trash.town key (without the .pub at the end). you want to send the ~/.ssh/trash.town.pub key.

to send your public ssh key to trash.town

  1. run cat ~/.ssh/trash.town.pub
  2. copy the output of text
  3. paste the output of text in an email addressed to signup@trash.town
  4. type your desired username into the subject line

tip: if the steps above didn't work for you, have a look at tilde.town's guide or tilde.town's key generator
